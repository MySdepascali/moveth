# movETH

## Installation

```
$ npm install
```

## Development

```
$ grunt
```

This starts a WebPack development server (with Hot Module Replacement) on http://localhost:8089/

## Watch loop

```
$ grunt dev
```

Starts a watch loop and run `jsxhint`, `jest` and `webpack` when a file is modified.

## Publish to Firebase

```
$ firebase deploy
```
